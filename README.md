 Fluxo de Uso

1. Primeira Visita
1. Acesse o site
2. Veja a home com apresentação
3. Clique em \"Agendar Agora\"
4. Modal de login aparece

2. Criar Conta
1. Clique em \"Criar conta\"
2. Preencha: Nome, E-mail, Senha
3. Clique em \"Criar Conta\"
4. Conta criada automaticamente!

 3. Buscar Profissional
1. Navegue pelas categorias
2. Use a busca se preferir
3. Veja os profissionais disponíveis
4. Clique em \"Ver Perfil\"

4. Ver Detalhes
1. Veja informações completas
2. Leia avaliações de outros usuários
3. Clique em \"Agendar Consulta\"

5. Fazer Agendamento
1. Selecione uma data
2. Escolha um horário disponível
3. Adicione observações (opcional)
4. Clique em \"Confirmar Agendamento\"
5. Veja a confirmação!

 6. Gerenciar Agendamentos
1. Vá em \"Meus Agendamentos\" (bottom nav)
2. Veja todos os agendamentos
3. Cancele se necessário

7. Ver Perfil
1. Acesse \"Perfil\" (bottom nav)
2. Veja suas estatísticas
3. Faça logout se quiser
